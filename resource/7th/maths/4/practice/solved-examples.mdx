## Example 1: Age Relationships
**Question:** Shabnam is 3 years older than Aftab. If Aftab's age is denoted by $a$, what is the expression for Shabnam's age? If Aftab is 23 years old, what is Shabnam's age?

**Solution:**
*   Aftab's age = $a$
*   Shabnam's age is 3 more than $a$.
*   Expression: **$s = a + 3$**
*   Substitute $a = 23$: $s = 23 + 3 = \mathbf{26 \text{ years}}$.

## Example 2: Matchstick Patterns
**Question:** Parthiv is making matchstick patterns placing "L" shapes next to each other. Each "L" has 2 matchsticks. What is the algebraic expression for the number of matchsticks required for $n$ shapes?

**Solution:**
*   1 "L" requires $1 \times 2 = 2$ sticks.
*   2 "L"s require $2 \times 2 = 4$ sticks.
*   $n$ "L"s require $n \times 2 = \mathbf{2n}$ sticks.

## Example 3: Shopping with Variables
**Question:** Ketaki supplies coconut-jaggery laddus. A coconut costs ₹35, and 1 kg jaggery costs ₹60. Write an algebraic expression to find the total amount paid for $c$ coconuts and $j$ kg of jaggery.

**Solution:**
*   Cost of coconuts = $c \times 35 = 35c$
*   Cost of jaggery = $j \times 60 = 60j$
*   Total Expression = **$35c + 60j$**

## Example 4: Splitting Areas
**Question:** A big rectangle is split into two smaller rectangles. The height is $v$. The width of the first is 4, and the width of the second is 3. Write an expression describing the area of the bigger rectangle in two ways.

**Solution:**
1.  **Method 1 (Total Area):** Find the total width $(4 + 3) = 7$. Area = height $\times$ total width = $v \times 7 = \mathbf{7v}$.
2.  **Method 2 (Sum of Smaller Areas):** Area of first rectangle = $4v$. Area of second rectangle = $3v$. Total Area = **$4v + 3v$**.

Because both methods describe the exact same rectangle, we prove that **$4v + 3v = 7v$**.

## Example 5: Distributive Property in Action
**Question:** Simplify the expression $4(x + y) - y$.

**Solution:**
*   Step 1: Use the distributive property to expand the bracket: $4(x + y) = 4x + 4y$
*   Step 2: Rewrite the full expression: $4x + 4y - y$
*   Step 3: Combine the like terms ($4y$ and $-y$): $4y - 1y = 3y$
*   Final Simplified Expression: **$4x + 3y$**

## Example 6: Checking Equivalency
**Question:** Are the expressions $5u$ and $5 + u$ equal to each other? Let's check by replacing $u$ with numbers.

**Solution:**
*   If $u = 2$:
    *   $5u = 5 \times 2 = 10$
    *   $5 + u = 5 + 2 = 7$ (Not equal)
*   If $u = 11$:
    *   $5u = 5 \times 11 = 55$
    *   $5 + u = 5 + 11 = 16$ (Not equal)

Therefore, $5u$ (5 times $u$) is **not equal** to $5 + u$ (5 more than $u$).
