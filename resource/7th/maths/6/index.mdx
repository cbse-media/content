## Quick Reference: The Rules of Parity

Parity refers to whether a number is **Even** or **Odd**. Here are the golden rules of parity arithmetic:

### Addition & Subtraction Rules
*   **Even** $\pm$ **Even** = **Even** (e.g., $4 + 2 = 6$)
*   **Odd** $\pm$ **Odd** = **Even** (e.g., $7 - 3 = 4$)
*   **Even** $\pm$ **Odd** = **Odd** (e.g., $8 + 3 = 11$)

### Algebraic Representation
If $n$ is any whole number ($1, 2, 3, \dots$):
*   An **Even** number is always a multiple of $2$. We write this as **$2n$**.
*   An **Odd** number is always one less (or one more) than an even number. We write this as **$2n - 1$** (or $2n + 1$).
